# Eye Tracking with MediaPipe Face Landmarker

Simple real‑time eye / iris tracking using your webcam and MediaPipe’s Face Landmarker (Tasks API).  
The script opens your camera and draws landmarks on both eyes.

---

## Requirements

- **Python**: 3.10 (recommended)
- **OS**: macOS (tested) with a working webcam
- **Python packages**:
  - `mediapipe` (0.10.x with Tasks API)
  - `opencv-python`
- **Model file**:
  - MediaPipe Face Landmarker `.task` model, downloaded from the official guide:  
    https://ai.google.dev/edge/mediapipe/solutions/vision/face_landmarker/python
